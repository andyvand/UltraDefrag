UltraDefrag это мощная программа для дефрагментации дисков. Может обрабатывать любые
системные файлы, включая реестр и файл подкачки. Работает быстро, эффективно и надежно.

ВОЗМОЖНОСТИ

Движок:

  • Поддержка FAT, exFAT и NTFS. (улучшено в 5.0.0)
  • Дефрагментация всех файлов, включая системные и мета-файлы NTFS. (улучшено в 5.0.0)
  • Полная оптимизация диска, которая обрабатывает все файлы. (улучшено в 5.0.3)
  • Быстрая оптимизация диска - обрабатывает в основном фрагментированные файлы. (улучшено в 5.0.3)
  • Оптимизация MFT. (улучшено в 5.0.1)
  • Исключение файлов по имени, числу фрагментов и размеру.
  • Ограничение времени обработки диска.
  • Запись отладочной информации в лог-файл. (только в 5.0.0)
  • Диски, нуждающиеся в проверке программой CHKDSK, исключаются из обработки. (только в 5.0.0)
  • Ещё...

Графический интерфейс:

  • Графическое отображение карты кластеров. (улучшено в 5.0.3)
  • Современный эргономичный дизайн. (улучшено в 5.0.3)
  • Высота списка дисков может настраиваться. (только в 5.0.0)
  • Индикация прогресса компактно встроена в список дисков. (улучшено в 5.0.0)
  • Можно задавать выключение компьютера (и другие полезные действия) после завершения дефрагментации. (улучшено в 5.0.2)
  • Есть перевод более чем на 58 языков мира. Легко добавить новый перевод. (улучшено в 5.0.3)
  • Ещё...

Утилита командной строки:

  • Полезна для автоматизации процессов и запуска дефрагментации по расписанию.
  • Используется для дефрагментации файлов и папок непосредственно из Проводника Windows.
  • Может отображать цветную карту кластеров!
  • Ещё...

Дефрагментация при загрузке Windows:

  • Полезна для обработки файлов, заблокированных системой. (улучшено в 5.0.0)
  • Есть интерактивный режим. (улучшено в 5.0.0)
  • Ещё...

Отчеты:

  • Генерируются как в удобном HTML, так и в простом текстовом формате. (улучшено в 5.0.3)
  • Включают полезную информацию о фрагментированных файлах. (улучшено в 5.0.4)
  • Ещё...

Установка:

  • Процесс установки очень удобен и прост. (улучшено в 5.0.0)
  • Возможна конфигурируемая автоматическая установка. (улучшено в 5.0.3)
  • Ещё...
