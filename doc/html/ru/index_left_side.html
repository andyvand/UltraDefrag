ВОЗМОЖНОСТИ

Движок:

  • Дефрагментация всех файлов, включая системные и мета-файлы NTFS. (улучшено в v5.0.0)
  • Полная оптимизация диска, которая обрабатывает все файлы. (улучшено в v5.0.0)
  • Быстрая оптимизация диска - обрабатывает в основном фрагментированные файлы. (только в v5.0.0)
  • Оптимизация MFT. (только в v5.0.0)
  • Исключение файлов по имени, числу фрагментов и размеру.
  • Запись отладочной информации в лог-файл. (только в v5.0.0)

Графический интерфейс:

  • Графическое отображение карты кластеров. (улучшено в v5.0.0)
  • Современный эргономичный дизайн. (только в v5.0.0)
  • Высота списка дисков может настраиваться. (только в v5.0.0)
  • Индикация прогресса компактно встроена в список дисков. (улучшено в v5.0.0)
  • Можно задавать выключение компьютера (и другие полезные действия) после завершения дефрагментации. (улучшено в v5.0.0)
  • Есть перевод более чем на 50 языков мира. Легко добавить новый перевод. (улучшено в v5.0.0)

Утилита командной строки:

  • Полезна для автоматизации процессов и запуска дефрагментации по расписанию.
  • Используется для дефрагментации файлов и папок непосредственно из Проводника Windows.
  • Может отображать цветную карту кластеров!

Дефрагментация при загрузке Windows:

  • Полезна для обработки файлов, заблокированных системой. (улучшено в v5.0.0)
  • Есть интерактивный режим. (улучшено в v5.0.0)

Отчеты:

  • Генерируются как в удобном HTML, так и в простом текстовом формате. (улучшено в v5.0.0)
  • Включают полезную информацию о фрагментированных файлах. (улучшено в v5.0.0)

Установка:

  • Процесс установки очень удобен и прост. (улучшено в v5.0.0)
  • Возможна конфигурируемая автоматическая установка. (только в v5.0.0)
