UltraDefrag is a powerful Open Source Defragmentation tool for the Windows Platform. It can defragment any system files including registry hives and paging file. Also one of the main goals of UltraDefrag is doing the job as fast and reliable as possible.

UltraDefrag 5.0.0 RC2 released [2011-12-24]

The new release candidate includes all the features of the final v5.0.0 release, which is scheduled to be released on December 31st, 2011. Since the last stable 4.4.0 release all parts of the program have been improved in many ways. The Change Log contains the full list of changes.

FEATURES

Driver:

  • Defragmentation of files and folders including system files and file system meta files. (enhanced in v5.0.0)
  • Full optimization of the drive, which processes all files despite of their fragmentation. (enhanced in v5.0.0)
  • Quick optimization of the drive, which mainly processes fragmented files. (new in v5.0.0)
  • MFT optimization, which processes the MFT only. (new in v5.0.0)
  • Filtering of files to process by name, fragment count and size.
  • User defined log file creation, which highly simplifies reporting bugs. (new in v5.0.0)
  • Read more...

Graphical User Interface:

  • Graphical display of the drives cluster map. (enhanced in v5.0.0)
  • Modern design with menus and tool bar. (new in v5.0.0)
  • The hight of the drives list can be adjusted. (new in v5.0.0)
  • The current progress and all important information are displayed in the drives list. (enhanced in v5.0.0)
  • A final action can be specified to be executed after processing the drives. (new in v5.0.0)
  • Color coding of the usage of the cluster map. (enhanced in v5.0.0)
  • Localization can be done by the user or by one of the included language packages. (enhanced in v5.0.0)
  • Read more...

Command Line Tool:

  • Useful for automation through scheduled tasks or batch scripts.
  • Integrated into Explorer's context menu.
  • Specify options through command line arguments or environment variables. (enhanced in v5.0.0)
  • On demand, display of the graphical cluster map.
  • Read more...

Boot Time Interface:

  • Useful to process locked system files. (enhanced in v5.0.0)
  • An interactive mode is available too. (enhanced in v5.0.0)
  • Read more...

Reports:

  • Available as plain text and HTML page. (enhanced in v5.0.0)
  • Includes all information about the fragmented files. (enhanced in v5.0.0)
  • Read more...

Installer:

  • Two configurations are available, full and without GUI. (enhanced in v5.0.0)
  • All features can be configured through command line arguments. (new in v5.0.0)
  • Selections of previous installations are retained. (new in v5.0.0)
  • Read more...
