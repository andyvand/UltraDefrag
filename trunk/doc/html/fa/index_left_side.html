NEWS

UltraDefrag 5.0.3 released [2012-02-26]
  • The GUI report options produce_html_report and produce_plain_text_report were added to the udreportopts.lua file; now the program generates text reports only when they are enabled explicitly.
  • A problem preventing language selection saving on installation was fixed.
  • Optional menu icons were added for Vista and more recent Windows editions; they can be enabled/disabled through the show_menu_icons GUI option.
  • A problem with the GUI cluster map drawing routine was fixed.
  • Some problems with the disk optimization algorithm were fixed.
  • Environment variables expansion support was added to the boot time shell.
  • The translation Wáray-Wáray has been added. Thanks to the translator.

The Change Log contains the full list of changes.

AVAILABLE INTERFACES

 

FEATURES

Driver:

  • Currently FAT, exFAT and NTFS are supported. (enhanced in 5.0.0)
  • Defragmentation of files and folders including system files and file system meta files. (enhanced in 5.0.0)
  • Full optimization of the drive, which processes all files despite of their fragmentation. (enhanced in 5.0.3)
  • Quick optimization of the drive, which mainly processes fragmented files. (enhanced in 5.0.3)
  • MFT optimization, which processes the MFT only. (enhanced in 5.0.1)
  • Filtering of files to process by name, fragment count and size.
  • Drive processing can be interrupted automatically after a defined amount of time.
  • User defined log file creation, which highly simplifies reporting bugs. (new in 5.0.0)
  • Drives needed to be checked by CHKDSK are excluded. (new in 5.0.0)
  • Read more...

Graphical User Interface:

  • Graphical display of the drives cluster map. (enhanced in 5.0.3)
  • Modern design with menus and tool bar. (enhanced in 5.0.3)
  • The hight of the drives list can be adjusted. (new in 5.0.0)
  • The current progress and all important information are displayed in the drives list. (enhanced in 5.0.0)
  • A final action can be specified to be executed after processing the drives. (new in 5.0.0)
  • Color coding of the usage of the cluster map. (enhanced in 5.0.2)
  • Localization can be done by the user or by one of the 58 included language packages. (enhanced in 5.0.3)
  • Read more...

Command Line Tool:

  • Useful for automation through scheduled tasks or batch scripts.
  • Integrated into Explorer's context menu.
  • Specify options through command line arguments or environment variables. (enhanced in 5.0.0)
  • On demand, display of the graphical cluster map.
  • Read more...

Boot Time Interface:

  • Useful to process locked system files. (enhanced in 5.0.0)
  • An interactive mode is available too. (enhanced in 5.0.0)
  • Read more...

Reports:

  • Available as plain text and HTML page. (enhanced in 5.0.3)
  • Includes all information about the fragmented files. (enhanced in 5.0.0)
  • Read more...

Installer:

  • Two configurations are available, full and without GUI. (enhanced in 5.0.0)
  • All features can be configured through command line arguments. (new in 5.0.0)
  • Selections of previous installations are retained. (enhanced in 5.0.3)
  • Read more...

 

SYSTEM REQUIREMENTS

Desktop PC: Windows NT 4.0, 2000, XP, Vista, 7

Server: Windows NT 4.0, 2000, 2003, 2008, 2008 R2

32-bit, 64-bit and Itanium Editions
